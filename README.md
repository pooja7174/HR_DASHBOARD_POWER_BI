📊 HR Analytics Dashboard – Power BI

An interactive HR Analytics Dashboard built in Power BI to analyze employee performance, workforce distribution, salary insights, and absenteeism trends across departments and locations.

This dashboard helps HR teams and management make data-driven decisions related to employee productivity, workforce planning, and performance monitoring.


🎯 Project Objective

The goal of this project is to transform raw HR data into meaningful business insights using Power BI visualizations, DAX calculations, and interactive filters.

The dashboard provides a complete overview of employee metrics including performance, absenteeism, salary distribution, and workforce demographics.

📁 Dataset Information

The dataset includes the following fields:

EmployeeID

EmployeeName

Department

Gender

Location

Salary

Performance Rating

Total records analyzed: 200 employees

🔍 Insights Derived

Some key insights from the dashboard:

Operations and Marketing departments have the highest number of employees.

40 employees are high performers, while 33 need improvement.

Workforce is male dominated (110 vs 90 female) but still relatively balanced.

Chennai and Bangalore show higher salary distribution.

Average absenteeism indicates potential attendance improvement opportunities.
